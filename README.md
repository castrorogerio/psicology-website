# Anabela Castro - Psicologia

Website profissional para Anabela Castro, psicóloga clínica.

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Desenvolvimento

```sh
# Instalar dependências
npm install

# Iniciar servidor de desenvolvimento
npm run dev
```

## Build

```sh
npm run build
```
